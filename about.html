While I've used R and Python quite extensively throughout grad school, I'm realizing that a large portion of mapping is done on the web. With that in mind, I decided to build my own website so I could play around with html, css, and web-mapping tools, api's and software such as leaflet, geojson and mapbox. As I learn more, I will post my projects here.

This site is hosted by GitHub Pages. As a learning exercise I took the html and css from Lyzi Diamon's site, which is hosted by Jekyll, and I altered it to suit my needs.
